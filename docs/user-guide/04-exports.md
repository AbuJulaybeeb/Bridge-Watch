# Exports

## Overview
The Export functionality allows you to extract data for offline analysis, accounting, or reporting. Note: You can refer to [Export Picker Flow](./export-picker-flow.md) for more details on the UI flow.

## Exporting Data
1. Navigate to any data table (e.g., Asset list, Alert history).
2. Click the **Export** button in the top right corner of the table.
3. Select your desired format:
   - CSV
   - JSON
   - Excel
4. Select the date range and columns to include.
5. Click **Download**.

## Automated Exports
Premium users can set up automated exports that deliver weekly or monthly reports to a specified email address or webhook.
